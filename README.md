# Veritone RPA Notes

A Robotic Process Automation bot that replaces the email-reply workflow of job applicants in Veritone Hire. Answers from applicants are automatically written into the Notes field of their application in AdCourier.

## Development

To test the Form URL locally, run:

```bash
npx serve .
```

This will start a local development server that serves the static site. You can then access the form through the provided local URL.
